# UML Runtime BPF Selftests Patches

Patches in this folder are applied after `../uml-veristat/`.

Put new patches here when they touch `tools/testing/selftests/bpf` or are only
needed for running BPF selftests or `test_progs` inside UML. Keep patches needed
by the base `uml-veristat` package and verifier corpus in `../uml-veristat/`.
